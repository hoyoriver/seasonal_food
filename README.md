# 🌱 제철 밥상

> 월별 제철재료와 지역별 특산물을 한눈에 볼 수 있는 웹 앱

**🔗 배포 주소:** https://hoyoriver.github.io/seasonal_food/

---

## 만든 과정

**날짜:** 2026-04-16
**도구:** Claude Code (AI 페어 프로그래밍)
**프롬프트:** 월별 제철재료를 알려주는 앱을 만들고 싶어. 지역별 특산물도 보여줄 수 있도록!

---

## 기능

- 📅 **월별 제철 탭** — 1~12월 선택 시 채소/수산물/과일 카드로 표시
- 📍 **지역 특산물 탭** — 제주, 경상도, 전라도, 충청도, 경기도, 강원도
- 현재 달 자동 선택
- 반응형 디자인 (모바일 지원)
- Pretendard 폰트 적용

---

## 파일 구조

```
seasonal-foods/

```

## 오늘 배운 것들

### 웹 앱의 3가지 구성요소
| 언어 | 역할 | 비유 |
|------|------|------|
| HTML | 뼈대 | 건물의 기둥과 벽 |
| CSS | 꾸미기 | 페인트, 인테리어 |
| JS | 동작 | 전기, 수도 |

### 서버 & 데이터베이스가 필요한 경우
- 로그인 / 회원가입
- 데이터 저장 (DB)
- 여러 사람이 같은 데이터를 주고받을 때
- 결제, AI API 연결

### 배포 방법
- **GitHub Pages** — 무료, 무기한, 저장소만 살아있으면 영원히 운영 가능
- **Netlify Drop** — 폴더 드래그 앤 드롭으로 1분 만에 배포
